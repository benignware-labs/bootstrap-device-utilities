Advanced Usage

Optimize css background-images for different device resolutions:

<div class="background">
  <div class="visible-device-lg visible-device-md visible-device-sm">
    <div style="background-image: url('images/flowers_lg.jpg');"> </div>
  </div>
  <div class="visible-device-xs">
    <div style="background-image: url('images/flowers_xs.jpg');"> </div>
  </div>
</div>
body,
html {
  width: 100%;
  height: 100%;
}
.background {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  z-index: -1;
}
.background > div > div {
  position: absolute;
  width: 100%;
  height: 100%;
  background-size: cover;
  background-repeat: no-repeat;
  background-position: center;
}
